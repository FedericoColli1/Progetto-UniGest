Descrizione dettagliata

Classe astratta che gestisce la creazione e il controllo dei JSON Web Token (JWT). Questa classe è responsabile dell'autenticazione degli utenti e della verifica dei token. Le funzioni che può svolgere sono:

  • Controllare la validità di un JWT ricevuto nell'header di autorizzazione.
  • Creare un nuovo JWT per un utente autenticato.

◆ control()

static control ( )
static

Controlla la validità di un JSON Web Token (JWT) presente nell'header 'Authorization' della richiesta HTTP. Estrae il token, lo decodifica utilizzando una chiave segreta e restituisce i dati del payload. In caso di token mancante o non valido, restituisce un errore HTTP 401.

Parametri
array$headersTutti gli header della richiesta HTTP.
string$provaIl valore dell'header 'Authorization' (per debug).
string$authHeaderIl valore dell'header 'Authorization'.
string$tokenTypeIl tipo di token (es. "Bearer").
string$tokenIl JWT effettivo.
object$datiI dati decodificati dal payload del JWT.
Restituisce
array|null I dati del payload del JWT decodificato in formato array associativo, o null se il token è mancante o non valido.

Crea un nuovo JSON Web Token (JWT) per un utente dopo l'autenticazione. Verifica le credenziali dell'utente (email e password) nel database e, se valide, genera un token con informazioni sul ruolo e i permessi dell'utente.

Parametri
string$emailL'indirizzo email dell'utente.
string$pwdLa password dell'utente.
string$secret_keyLa chiave segreta utilizzata per firmare il JWT.
Database$dbconnOggetto Database per la connessione al DB.
string$queryQuery SQL per recuperare i dati dell'utente.
array$resultsIl set di risultati della query.
array$rowL'array associativo contenente i dati dell'utente dal database.
array$payloadIl payload del JWT, contenente informazioni sull'utente.
string$jwtIl JSON Web Token generato.
array$userArray contenente il token da restituire.
array$erroreArray di errore in caso di credenziali non valide.
Restituisce
void Restituisce il token JWT in formato JSON o un messaggio di errore HTTP.
