static DownloadFile ( )
static

Gestisce il download di un file specifico richiesto tramite parametri GET. Recupera l'ID della pratica, il numero del passaggio e il nome del file dai parametri della richiesta. Verifica l'esistenza del file sul server e, se presente, imposta gli header HTTP appropriati per il download e invia il contenuto del file al browser. In caso il file non esista, restituisce un errore HTTP 404.

Parametri
string$_GET,['IdPratica']L'ID della pratica a cui il file appartiene.
string$_GET,['NPassaggio']Il numero del passaggio a cui il file è associato.
string$_GET,['File']Il nome del file da scaricare.
string$praticaL'ID della pratica estratto da $_GET.
string$passaggioIl numero del passaggio estratto da $_GET.
string$fileIl nome base del file estratto da $_GET.
string$pathIl percorso completo del file sul server.
Restituisce
void Il file viene scaricato direttamente dal browser o viene restituito un messaggio di errore.
