Descrizione dettagliata

Classe che rappresenta un utente nel sistema. Implementa l'interfaccia UserInterface. Le funzioni che può svolgere sono:

  • Inizializzare un utente recuperando i suoi dati dal database.
  • Restituire i valori dei suoi attributi (ID, Nome, Mail, Pwd, DataCreazione, Jolly).
  • Ottenere le pratiche associate all'utente.
  • Ottenere i documenti relativi a una specifica tipologia di pratica.
  • Aggiungere una nuova pratica.

◆ getDocumenti()

getDocumenti ( $tipologia)

Recupera la lista di documenti associati a una specifica tipologia di pratica. La query cerca la colonna 'Passaggi' (presumibilmente contiene un JSON con i dettagli dei passaggi inclusi i documenti) e poi tenta di estrarre i "Documenti". Nota: la riga $documenti = $rows["Documenti"]; sembra fare riferimento a una colonna "Documenti" che non è selezionata nella query "SELECT Tipologia.Passaggi". Potrebbe essere un errore logico o una colonna omessa nella query.

Parametri
string$tipologiaLa tipologia di pratica di cui si vogliono ottenere i documenti.
Database$dbconnRichiama oggetto Database\query() per creare una connessione con il db.
string$queryUna query SQL per selezionare i passaggi della tipologia.
mysqli_result$resultsIl set di risultati della query.
array$rowsUn array associativo contenente i dettagli dei passaggi.
Restituisce
mixed La lista dei documenti o un valore indefinito se la colonna "Documenti" non esiste o la query fallisce.
